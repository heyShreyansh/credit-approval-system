POST /register/
Register a new customer in the system

Required Fields:

  • first_name (string, max 100 chars)
  • last_name (string, max 100 chars)
  • age (integer, 18-100)
  • monthly_income (decimal, max 10 lakhs)
  • phone_number (string, 10-15 digits)
{ "first_name": "John", "last_name": "Doe", "age": 30, "monthly_income": 50000, "phone_number": "1234567890" }
POST /check-eligibility/
Check if a customer is eligible for a loan

Required Fields:

  • customer_id (integer)
  • loan_amount (decimal, max 1 crore)
  • interest_rate (decimal, 0-50%)
  • tenure (integer, 1-360 months)
{ "customer_id": 1, "loan_amount": 100000, "interest_rate": 12.0, "tenure": 12 }
POST /create-loan/
Create a new loan if the customer is eligible

Required Fields:

  • customer_id (integer)
  • loan_amount (decimal, max 1 crore)
  • interest_rate (decimal, 0-50%)
  • tenure (integer, 1-360 months)
{ "customer_id": 1, "loan_amount": 100000, "interest_rate": 12.0, "tenure": 12 }
GET /view-loan/{loan_id}/
View details of a specific loan

URL Parameters:

  • loan_id (integer) - The ID of the loan to view
GET /view-loan/1/
GET /view-loans/{customer_id}/
View all active loans for a specific customer

URL Parameters:

  • customer_id (integer) - The ID of the customer
GET /view-loans/1/
